Saving archive data

This procedure describes how to save job log and output archived data to a text file.

To save archive data:

  1. Right-click the service that contains the jobs you want to search for and select Browse Jobs.
  2. Perform a search, as described in Searching for archive data.
  3. In the Properties pane, click the Log or Output tab and then click Save.
  4. Browse to a location where you want to save the archived data, and then click Save.

    The archived data is saved in the specified location.
